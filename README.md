# SmartStudyPlanner# Smart Study Planner

## Description
Smart Study Planner is a Python console application that helps students organize their study tasks.  
The application allows users to add tasks, manage deadlines, calculate priorities, and view statistics.

## Features
- Add new study tasks
- View all tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Save tasks in a JSON file
- Generate a recommended study plan
- Calculate priority based on deadline, difficulty, and estimated hours
- Show task statistics
- Validate user input

## Technologies Used
- Python
- JSON
- datetime module
- Visual Studio Code

## How to Run
1. Open the project folder in Visual Studio Code.
2. Make sure `main.py` and `tasks.json` are in the same folder.
3. Open the terminal.
4. Run:

python main.py
